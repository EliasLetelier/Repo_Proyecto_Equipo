# Documentación de decisiones
#### Los datos que se analizaron es sobre la información de cada partido entre Universidad de Chile y Colo-Colo, fueron en las páginnas ProMedios y Transfermark. Estas páginas muestran la información sobre el día en donde se realizó cada partido, la localía, el estableimiento, el árbitro, los espectadores, las tarjetas, faltas, cambios y resultado de cada uno.
#### Para hacer la limpiza de datos, se ocuparon los datos que podrían servir para el proyecto y la base de datos inicial realizada en la entrega anterior. Por lo que se tomaron las estadísticas de la fecha de los partidos, cuál equipo fue local, la competición en la cual se enfrentaron, el establecimiento y el árbitro a cargo del momento.
#### Estas fuentes están ordenados de izquierda a derecha en la base de datos con el siguiente orden:
#### - Fecha
#### - Colo-Colo local
#### - U de Chile local 
#### - Torneo Nacional
#### - Copa Chile
#### - Super Copa
#### - Árbitro
