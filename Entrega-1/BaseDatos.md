# Base de Datos Inicial
## Plan de construcción (Si no existe completa)
#### Dado que la ANFP no tiene una base de datos única descargable de faltas desde 1990, construiremos una base mediante inscripción manual y web scraping de sitios estadísticos (Soccerway, WorldFootball) y archivos de prensa.
#### •	Fuentes: Sitio oficial ANFP, base de datos histórica de SofaScore,  sitio oficial de ESPN, Sofascore, ProMedios
#### •	Método: Extracción manual de datos de planillas antiguas y uso de scripts para datos modernos.
#### •	Columnas/Variables:
#### o	Fecha (DD/MM/AAAA)
#### o	Torneo (Apertura, Clausura, Nacional y Copa Chile)
#### o	Faltas_ColoColo 
#### o	Faltas_UdeChile 
#### o	Total Faltas 
#### o	Tarjetas Amarillas 
#### o	Tarjetas Rojas 
#### o	Arbitro 
## Ficha Metodológica
#### •	Autor: Grupo de investigación (Uds.) basado en datos de la ANFP.
#### •	Contenido: Estadísticas de disciplina de partidos oficiales entre Colo-Colo y U. de Chile (2000-2025).
#### •	Pertinencia: Es fundamental porque la "falta" es la unidad básica para medir la interrupción violenta del juego.
#### •	Metodología: Recopilación a partir de informes arbitrales de la ANFP y validación cruzada con crónicas de prensa deportiva para datos anteriores a 2010.
